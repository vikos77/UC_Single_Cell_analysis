# Week 2: Quality Control

## Overview
Quality control and filtering of single-cell RNA-seq data.

## QC Metrics
- Mitochondrial gene percentage
- Number of genes per cell
- Total counts per cell

## Filtering Thresholds
- MT% < 20%
- Min genes: 200
- Max genes: 6000
- Min cells per gene: 3
